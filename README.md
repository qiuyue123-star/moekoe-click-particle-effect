# MoeKoeMusic 全局鼠标点击特效插件
## 插件基础信息
- 插件ID：moekoe-click-particle-effect
- 版本：v1.0.0
- 作者：篍约
- 创作时间：2026-07-14
- 适配客户端：MoeKoeMusic 全版本

## 功能介绍
1. 总开关控制：可一键启用/关闭全局点击特效，默认关闭不占用性能
2. 三种可选点击粒子效果
   - 渐变紫粒子：渐变紫色圆形飘散粒子
   - 浅粉白蝴蝶结：粉白横向蝴蝶结样式动画
   - 水滴：上下渐变水滴下落粒子
3. 独立弹窗设置面板，内置插件作者、版本、简介信息
4. 全页面全局生效，播放器、设置页、插件市场全部支持

## 安装方式
1. 将本工程所有文件打包为zip压缩包
2. 打开MoeKoeMusic客户端 → 左侧「插件」页面
3. 点击右上角「安装插件」，选中zip文件完成导入
4. 插件列表点击「设置」打开特效配置面板
